# 公众号排版生成器 v3

一个可视化微信公众号排版工具，支持 40+ 组件，一键复制到公众号后台。

## 在线使用

👉 [https://we1zheng.github.io/wechat-editor](https://we1zheng.github.io/wechat-editor)

## 功能特点

- 🎨 **40+ 精美组件**：封面、标题、段落、引用、数据卡片、时间轴、FAQ 等
- 📱 **实时预览**：所见即所得编辑体验
- 📋 **一键复制**：生成的排版可直接粘贴到公众号后台
- 💾 **本地运行**：无需后端，纯前端 HTML 文件

## 本地使用

直接在浏览器中打开 `index.html` 即可使用。
